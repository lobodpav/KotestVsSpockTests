# Kotlin Training

Compares 
[Kotest](https://kotest.io)
and
[Spock](https://spockframework.org)
testing frameworks.
                          
## Requirements
* JDK 17

## Frameworks and tools in use
* Gradle 7
* Groovy 3
* Kotlin 1.7
* Spock 2

## Build it, test it

```shell
./gradlew build
```
           
## Run it

```shell
./gradlew run
```
